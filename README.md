# EmpresaLab
EmpresaLab é uma plataforma de simulação empresarial que automatiza o controle financeiro de empresas, incluindo DRE, fluxo de caixa, estoque e balanço gerencial. Voltado para uso acadêmico, permite que alunos registrem dados e que administradores acompanhem e analisem os resultados em tempo real.
